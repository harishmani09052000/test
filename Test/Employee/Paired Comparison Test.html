Welcome to Paired Comparison test!

        The number of samples in this test is fixed. In this test there are two samples and both the samples are different, it is presented simultaneously to the panel. The judge or panelist is asked to determine whether there is a perceivable difference. A positive answer is required even if it is a guess since all three samples are unknown, the chance of probability of placing the sample in a certain order is one third,

        In Paired Comparison test there are two samples one is a standard or control the other is an experimental one. Paired comparison test is used to determine if the difference between the two samples can be discriminated. The panelist is asked to identify if there is a difference. If yes, they are asked to indicate the degree of difference or intensity. All samples should be uniform as possible with respect to other characteristics. If one sample has a different colour from the paired samples, then the panelist need not even taste to make correct identification

        The Paired Comparison test can be extended to identify the intense sample. After identifying if there is a difference between the samples. The panelist is asked to identify which attribute, such as sweetness, acidity, hardness, colour etc. is responsible for this difference. In evaluation the results for degree of difference are taken into consideration. This step is done to determine a perceivable difference or a flavour difference. Paired comparison is typically used in comparing new and old processing techniques, change of ingredients in a product, etc

The type of panelist to be employed for this test can to be a trained or untrained panelist
