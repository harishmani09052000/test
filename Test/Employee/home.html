Sensory Eval Pro


Sensory Eval Pro is a comprehensive Software that incorporates essential primary sensory evaluation tests. This tool assists in arranging a sensory evaluation tailored to your specifications. It generates a link and code that can be distributed to panelists to take up the sensory evaluation. To uphold test integrity, each panelist receives uniquely coded samples. Panelists start by entering the sample code, sharing their preferences, and noting observations. The software simplifies each test, providing clear explanations for easy comprehension and operation.
